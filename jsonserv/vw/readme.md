#  Промежуточные модели для доступа к данным через представления (view)
Все модели должны иметь указание об отключении управления таблицами и указание view как db_table
```python
class Meta:
    managed = False
    db_table = 'vw_links'
```
Необходимые представления создаются предварительно с помощью скриптов из каталога scripts
