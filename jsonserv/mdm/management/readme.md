# Команды в составе поставки

## Импорт данных из файла Excel в rawrows

```cmd
python manage.py import_raw 1 vlmk.json
```
где
* 1 - идентификатор сессии, от которой ведется загрузка
* vlmk.json - имя файла с настройками загрузки (в каталоге imp_settings)

Файл с данными должен находится в каталоге temp
