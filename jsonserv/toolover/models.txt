### **externallibraryreference** Ссылки на внешние библиотеки
Связка свойств и внешних библиотек, их определяющих.
*property Ссыока на свойство
*external_id Идентификатор библиотеки
*library_type Тип библиотеки
*description Описание
### **gtcpackage** Загруженные GTC-пакеты
Свойства всех загруженных в базу данных GTC-пакетов:
*gtc_package_id Уникальный идентификатор пакета
*supported_gtc_generic_versions"Supported GTC generic versions
*vendor_hierarchy_version Vendor hierarchy version
*vendor_name Vendor name. Вендор, предоставивший пакет
*vendor_acronym Vendor acronym. Краткое обозначение вендора
*gtc_package_creation_date GTC package creation_date. Дата создания GTC-пакета
*logo ссылка на файл логотипа
*download_security Download security
*vendor_system_version Vendor System version
*short_description Short description. Краткое описание
