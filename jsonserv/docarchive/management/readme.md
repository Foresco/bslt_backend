# Команды в составе поставки

## Проверка наличия файлов в архивах
Если по каким-то причинам информация о наличии файла в архиве не соответствует действительности, 
то получить список таких файлов можно командой
```cmd
python manage.py check_arc_files
```
список найденных ошибочных ссылок сохраняется в каталоге с логами под именем
wrong file links дата-время.log
Возможные дополнительные параметры:
* --delete_rows Удалять неверные ссылки из базы данных


## Перенос в архив всех удаленных файлов
Если необходимо переместить удаленные файлы из архива в другое место, 
то нужно выполнить команду
```cmd
python manage.py move_deleted_arc_files /opt/www/basalta/reservstorage --archive_name filestorage
```
где:
* /opt/www/basalta/reservstorage - каталог для перемещения
* --archive_name дополнительный параметр для отбора обрабатываемого архива
