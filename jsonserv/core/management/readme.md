# Команды в составе поставки

## Обновление ключевого кода у всех экземпляров сущности
Если по каким-то причинам произошел сбой в формировании ключевого кода, то привести
его значения в порядок можно командой **recast_head_codes**
```cmd
python manage.py recast_head_codes 1 partobject
```
где
* 1 - идентификатор сессии, в которой ведется обновление
* partobject - имя (идентификатор) обрабатываемой сущности

Внимание! Перед выполнением надо отключить контроль уникальности обозначений
в обрабатываемой модели (обычно в методе Entity.save или дочернем)

## Экспорт объектов в формат json
Экспорт объектов определенной модели
```cmd
python manage.py export_json Classification
```
где
* Classification - наименование модели


## Закрытие всех сессий всех пользователей
Если нужно закрыть все активные сессии. Лучше делать это когда никто не работает.
```cmd
python manage.py expire_all_sessions
```
