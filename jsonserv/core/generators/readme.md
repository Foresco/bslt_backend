# Генераторы
В каталоге находятся генераторы обозначений для разных сущностей.
Необходимость использования генератора указывается в свойствах типа (модель core.EntityType.generator)

Каждый генератор должен наследовать от класса Generator
и реализовывать функцию gen_new_code

В общем случае набор генераторов уникален для каждого проекта
